- Папка с проектом:
1. templates - папка для шаблонов(моделей)
2. manage.py - папка для управления проектом, с его помощью будем выполнять команды
- Папка с настройками:
1. asgi - он нужен для ассинхронных веб-серверов и приложений
2. wcgi - для синхронных веб-серверов и приложений
3. urls - это главный файл для наших url проекта, туда будем подключать url наших приложений
4. settings - это файл со всеми настройками проекта:
   1. BASE DIR - построение пути и корни нашего проекта
   2. DEBUG - буливо значение которое включает и выключает режимы отладки проекта
   3. ALLOWED HOSTS - не используется при включённой отладке и запуске тестов, 
   добавляем домен сайта в эту настройку
   4. INSTALLED APPS - настройка которую мы будем изменять во всех наших проектах, она указывает Django 
   какие приложения активны в нашем проекте
   5. MIDDLE WARE - список подключённых промежуточных слоёв
   6. ROOT_URLCONF - указывает на пайтон модуль, который указывает корневые шаблоны urls приложения
   7. DATABASES - представляют собой словарь содержащий настройки для всех баз данных проектов
   8. LANGUAGE_CODE - подключает язык сайта
   9. TIME_ZONE - USE_TZ - указывает Django необходимость поддержки временных зон
